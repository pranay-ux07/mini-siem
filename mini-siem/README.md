# 🛡️ Mini SIEM - Log Analysis & Intrusion Detection System

## 📌 Overview
This project is a Python-based Mini SIEM system that analyzes authentication logs to detect suspicious activities such as brute-force attacks and unauthorized access attempts.

## 🚀 Features
- Detects brute-force attacks based on failed login attempts
- Identifies suspicious IP activity
- Generates a CSV security report
- Modular and clean code structure

## 🛠️ Technologies Used
- Python
- Regular Expressions (Regex)
- File Handling
- CSV Reporting
